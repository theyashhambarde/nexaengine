# 🌌 Nexa — Direct your curiosity.

**Nexa** is a premium, AI-powered answering engine designed to bridge the gap between search and research. Built with modern web technologies, it provides real-time, cited answers with a focus on speed, depth, and a stunning user experience.

![Nexa UI Preview](https://via.placeholder.com/1200x600/0a0a0a/ffffff?text=Nexa+AI+Answering+Engine)

---

## ✨ Features

- **🔍 Multi-Mode Research**: Choose your depth of discovery.
  - **Flash**: Lightning-fast answers for immediate results.
  - **Smart**: Intelligent processing that balances detail and velocity.
  - **Deep**: Exhaustive analysis and multi-step research for complex tasks.
- **⚡ Real-time Citations**: Every answer is backed by live web sources with clear citations.
- **🧠 Visual Thinking Process**: Watch the AI "think" in real-time with our custom-animated brain-circuit indicator.
- **🌒 Premium Dark Theme**: A custom-designed, deep onyx black interface optimized for focus.
- **📱 Responsive Excellence**: Fully optimized for desktop, tablet, and mobile devices.
- **📁 Document Intelligence**: Upload and search through your own documents (PDF, Docx, etc.).

---

## 🚀 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Database**: [Drizzle ORM](https://orm.drizzle.team/) with [SQLite](https://www.sqlite.org/)
- **UI Components**: [Headless UI](https://headlessui.com/), [Lucide React](https://lucide.dev/)
- **AI Models**: Support for Gemini, OpenAI, and local Ollama instances.

## 🛠️ Getting Started

### Prerequisites

- Node.js 20+
- pnpm (recommended) or npm

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/theyashhambarde/nexaengine.git
   cd nexaengine
   ```

2. **Install dependencies:**
   ```bash
   pnpm install
   # or
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add your keys:
   ```env
   # API Keys
   GOOGLE_GENERATIVE_AI_API_KEY=your_key
   # Add other keys as needed
   ```

4. **Run the Development Server:**
   ```bash
   pnpm dev
   # or
   npm run dev
   ```

5. **Build for Production:**
   ```bash
   pnpm build
   pnpm start
   ```
